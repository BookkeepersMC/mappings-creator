# mappings-creator

## Usage
**COMING SOON**
Using on the Gradle Plugin Portal!

This plugin creates a project for *YOU* that allows you to layer any mappings into loom! (Fabric/Quilt/Arch Loom)

Anyone can create mappings, and now it's easier than ever!
This Plugin auto-creates a mappings project, that you can edit, and re-publish! Uses enigma by @QuiltMC

Defaults to Mojang mappings & you can rename!
